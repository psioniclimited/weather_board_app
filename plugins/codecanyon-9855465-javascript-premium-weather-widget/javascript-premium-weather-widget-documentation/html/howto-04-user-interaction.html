User Interaction

The widget is capable of responding to some user actions in order to change locations, change measure units or show/hide weather forecast.
Most of the included themes reacts in a similar way as described below.

Location Change

In order to change the location, just click the location label and it will become editable. Press Enter to confirm the new location, or press Esc or click away to cancel.

Changing location

Measure Units

To change the measure units for temperature, wind speed and pressure click the temperature label.

Changing measure units

Weather Forecasts

In the case the current theme does not display the forecast, just click the weather icon to bring it up. If the theme is designed to show the forecast, the weather forecast will nicely fade-in.

Note:
Not all the themes are designed to display weather forecasts.

Widget Refresh

In order to refresh the weather data, most of the themes support this by clicking the weather data or forecast data. Some of them (the ones that do not have forecast) refreshes when clicking the weather icon while others might have a refresh button.

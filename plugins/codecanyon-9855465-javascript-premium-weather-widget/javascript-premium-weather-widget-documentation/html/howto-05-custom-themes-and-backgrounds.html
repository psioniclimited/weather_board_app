Custom Themes and Backgrounds

Custom Themes

The themes are HTML files consisting of a layout template and some CSS styling.
In order to create a custom theme, it's better if you start from an existing one. Just pick the theme that is the most closed to your desired design and start tuning it up.
All the themes are located under the javascript-premium-weather-widget/bins/themes directory, so your theme should be copied into that location too.

Themes directory

In order to use the new 'my-custom-theme.html' your widget directive should look like: 

<div ng-app="wl-ng-weather"
    wl-ng-weather
    theme="my-custom-theme"
    background="dynamic"></div>
Note:
Please notice that the .html extension should be skipped in the directive attributes.

Custom Backgrounds

In case that the background attribute is set to "dynamic", the background of the weather widget will be set to a picture corresponding to the current weather conditions.
Those pictures are located under the javascript-premium-weather-widget/bins/assets directory and they are named accordingly by the formula: 'bg-condition-day|night'.
The package provides by default royality free pictures for the dynamic backgrounds but they can be replaced by some more beautiful and personal not-free pictures if desired.
